# Sandy Bains — Real Estate Website

Marketing website for **Sandy Bains, REALTOR®** with HomeLife/Silvercity Realty Inc., Brokerage in Brampton, Ontario.

## What's here

- `index.html` — the complete single-page site (styles and scripts included)
- `sandybains/portrait.jpg` — portrait photo
- `sandybains/fonts/` — self-hosted web fonts (Fraunces and Instrument Sans)

The site is fully static with no build step and no external dependencies, so it can be hosted anywhere.

## Publishing with GitHub Pages

1. In this repository, go to **Settings → Pages**
2. Under *Build and deployment*, choose **Deploy from a branch**
3. Select branch **main** and folder **/ (root)**, then save

The site will be live at `https://<username>.github.io/Sandy/` within a couple of minutes. A custom domain such as `www.sandybains.ca` can be connected on the same settings page.

## Editing content

Everything is in `index.html`:

- **Contact details** appear in the hero card, the contact section and the footer
- **Client testimonials** are placeholders — replace them with real client quotes
- **Portrait** — to use a professional headshot, replace `sandybains/portrait.jpg` (portrait orientation, roughly 3:4)
